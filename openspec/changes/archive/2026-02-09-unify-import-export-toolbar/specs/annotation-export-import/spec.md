## MODIFIED Requirements

### Requirement: Import and export controls in toolbar
The system SHALL provide Import and Export dropdown menus in the toolbar. The Import dropdown SHALL contain options for importing an image or a JSON file. The Export dropdown SHALL contain options for exporting as MEI or JSON.

#### Scenario: Toolbar displays Import and Export dropdowns
- **WHEN** the toolbar is rendered
- **THEN** an "Import" dropdown button and an "Export" dropdown button SHALL be visible as the first group in the toolbar
